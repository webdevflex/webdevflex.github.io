Новый Microsoft Edge нельзя будет удалить из Windows 10

Пользователи Windows Insider обнаружили одну особенность нового Microsoft Edge на Chromium. Те, кто скачал стабильную версию нового Microsoft Edge и установил ее на текущую тестовую сборку Windows 10 под кодовым именем 20H1, не могут удалить браузер из системы.

Через меню с установленными приложениями больше нельзя удалить браузер, как и не отдельного компонента для запуска удаления в папке с ним. А учитывая то, что после установки браузера, классический Microsoft Edge система скрывает как можно сильнее, то нетрудно догадаться, что у Microsoft есть четкое видение по поводу нового браузера. Старый браузер останется скорее, как компонент для запуска старых приложений, а новый будет браузером по умолчанию.

Стоит оговориться, что стабильная версия Microsoft Edge на Chromium утекла в сеть раньше времени, но на официальном канале распространения она пока недоступна.

Что же касается самого браузера, то его тестирование началось в апреле, а после условного релиза, его продолжат обновлять в таком же темпе. А Windows 10 с обновлением 20H1 выйдет весной 2020 года, тестирование данного обновления началось еще в феврале.

В целом Microsoft Edge на Chromium оставляет после себя неоднозначное впечатление. С одной стороны, перейдя на Chromium он лишился своей индивидуальности и собственного движка.

С другой стороны, он стал браузером, которым можно пользоваться. Сайты открываются без искажений и найти неработающий именно в Edge не удалось.

Производительность тоже радует, а также есть все необходимые возможности, будь то синхронизация закладок, паролей и других данных, мобильная версия браузера и другое.

Если лень или нет времени настраивать систему, то Microsoft Edge можно комфортно использовать и даже рекомендовать другим, чего о браузерах Microsoft нельзя было сказать больше десяти лет.
